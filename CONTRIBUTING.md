# Contributing to MermaidFlow

Thanks for your interest! Here's how to get started:

## Setup
1. Fork the repo
2. `npm install`
3. `npm run dev`

## Areas That Need Help
- Better Mermaid syntax highlighting in CodeMirror
- Batch export (multiple diagrams → ZIP)
- localStorage diagram history
- PDF export
- More diagram templates

## PR Guidelines
- Keep PRs focused (one feature or fix per PR)
- Add a short description of what changed and why
- Test in both light and dark mode
