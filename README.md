# MermaidFlow 🧜

> Free, open-source, web-based Mermaid diagram editor with unrestricted PNG/SVG export.

![License](https://img.shields.io/badge/license-MIT-green)
![Vite](https://img.shields.io/badge/built%20with-Vite-646CFF)

## Features

- 📝 Live editor with syntax highlighting (CodeMirror 6)
- 🎨 5 built-in Mermaid themes
- ⬇️ PNG export at 1x, 2x, 3x, 4x resolution — **no sign-up, no paywall**
- ⬇️ SVG export
- 🔗 Shareable permalink (diagram encoded in URL, no backend)
- 📦 Diagram type templates (flowchart, sequence, ER, Gantt, pie, class)
- 🌙 Dark mode
- 💾 Zero backend — runs entirely in your browser

## Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Build for production
npm run build
```

## Tech Stack

| Tool | Purpose |
|------|---------|
| React + Vite | Frontend framework |
| mermaid.js | Diagram rendering |
| CodeMirror 6 | Code editor |
| Tailwind CSS | Styling |
| lz-string | URL compression for sharing |
